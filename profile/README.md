## 🎯 프로젝트 소개

**AI 점원형 음성 대화 키오스크 시스템**은 기존 터치 기반 키오스크의 한계를 극복하고, 음성 대화를 통해 누구나 쉽게 사용할 수 있는 차세대 키오스크 솔루션입니다.

### ✨ 주요 기능
- 🎤 **음성 인식 주문** - 자연스러운 대화로 메뉴 주문
- 🤖 **AI 추천** - 사용자 맞춤형 메뉴 추천
- 👁️ **얼굴/포즈 감지** - MediaPipe 기반 사용자 인식
- 🔊 **음성 응답** - TTS를 통한 친근한 안내
